Technology

Restful API is a popular standard applied in designing APIs for web applications, aiming to optimize resource management. Instead of focusing on specifying how applications should behave, Restful API concentrates on managing system resources such as text files, images, audio, video, and dynamic data by formatting and transmitting their states through the HTTP protocol.

Security

JWT (JSON Web Token) is an open standard used to create and transmit authentication information securely and reliably between parties. JWTs are commonly used for user authentication and access authorization in web applications and web services. Key features of JWT include mobility and convenience, allowing for easy transmission of authentication information between parties without the need for storage in databases or stateful memory.
